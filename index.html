Robot 1 : BMI Queue Process

UiPath Orhestrator Body Mass Index (BMI) Calculator Website MS Excel

In UiPath queue terminology, the concepts of Dispatcher and Performer are often used, in transactional process automation. This define how queue items are added and processed.
Dispatcher - Adds items to the queue. Feeds the robot with work to process. Reads input data (e.g., Excel, database, web forms).
Performer - Processes items from the queue.Performs the actual business process on each transaction.
Set Transaction Status to mark each item as: Successful, Failed (with retry if enabled) Business Exception.

Robot 2: Invoice Processing Automation

UiPath ReFramework UiPath Document Understanding CRM - Python Streamlit

Alfred is an automation bot that streamlines Finance & Accounting invoice processing, targeting a 70% reduction in processing time and improved accuracy.
